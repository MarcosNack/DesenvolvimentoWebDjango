Questão 2 – O código abaixo é o exemplo de uma sintaxe básica de um código CSS que possa ser utilizado dentro do Framework Django:

h1 {

font-family: times new roman;

font-size: 30px;

background-color #0000FF;

}

#formato_paragrafo {

font-family: arial;

font-size: 10px;

background-color #7FFF00;

}

Neste sentido o exemplo apresentado ilustra o trecho de código CSS externo, que possui uma seção única constituída de seletores, e estes de um corpo de declaração que, deve conter propriedades específicas ao elemento HTML. Diante disso, dentro do HTML os marcadores necessários para aplicar o estilo externo:

a) É necessário se utilizar de marcadores de elementos HTML acrescidos do atributo style e o nome da classe quando necessário

b) É necessário utilizar apenas de marcadores HTML, quaisquer que eles sejam, desde que no código CSS for selecionados adequadamente

c) É necessário utilizar o atributo style para cada elemento HTML e dentro das TAGs <head> e </head> na seção de cabeçalho definir as localizações das classes

d) Somente é preciso definir a localização das classes nas TAGs <script> e </script> que o navegador subentende que as regras serão aplicadas para cada elemento HTML

e) Deve-se avisar ao navegador dentro das TAGS <body> e </body> a localização do arquivo externo CSS, e para cada elemento HTML atribuir uma classe com o identificador ID.

Questão 2 – Alternativa B.

quando o código CSS é externo é necessário definir regras no arquivo CSS a partir de seletores para cada elemento HTML e definir as classes específicas como mostra o código exemplo do enunciado, com o símbolo sustenido ou jogo da velha (#). Além de realizar a seleção dos elementos que serão estilizados, é necessário definir a localização do arquivo dentro da seção cabeçalho no código HTML.

Questão 3 – A linguagem JavaScript deve ser definida como uma linguagem que é interpretada e tem seu funcionamento dependente de funcionalidades hospedadas no navegador. Isso significa que a linguagem JavaScript é uma linguagem:

a) Compilável pelo navegador de internet.

b) Usada para rodar do lado do cliente.

c) Usada para rodar do lado do servidor.

d) Script que trabalha obrigatoriamente com tags.

e) Se utiliza de tags seletoras equivalente ao CSS.
--------------------------------------------------
Questão 3 – Alternativa B.

A linguagem JavaScript não é compilável, e sim interpretada pelo navegador de internet. Não precisa de uma aplicação específica para executar um arquivo de código executável; por isso, a linguagem JavaScript é denominada linguagem script. A linguagem JavaScript é uma linguagem trabalhada do lado cliente, na camada front-end, ou seja, ela é interpretada e funciona a partir de funcionalidades hospedadas no navegador. Assim, na maior parte das vezes, o JavaScript não se utiliza de funcionalidades de serviços de um servidor.  Contudo, mesmo sendo um linguagem script, é uma linguagem de programação orientada ao objeto e que trabalha com comandos e instruções parecidas com o Java, mas não define scripts HTML nem para o CSS.

a) Roteamentos URL no arquivo url.py

b) Templates no arquivo de extensão .html

e) Parametrizações no arquivo manage.py
