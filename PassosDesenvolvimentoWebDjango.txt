1) Instalação do Python - OK

2) Criar e Ativar ambiente virtual
  a) venvwebdjango
  
3) Instalação do Pacote Django (PIP)
  a) pip install Django
  
4) Criação do Projeto
  a) Comando para criar o projeto Django django-admin startproject <nome do projeto>
    --> django-admin startproject projeto_aula_desenv_web
  
  b) Executar o projeto
    --> py manage.py runserver
